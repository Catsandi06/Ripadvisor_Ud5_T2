# RipAdvisor

##Descripcion

Aplicacion de escritorio para gestionar un sistema de ranking de restaurantes.

##Requisitos

- Java JDK 8 o superior

##Como ejecutar

javac -d . src/net/salesianos/*.java src/App.java
java net.salesianos.App

##Funcionalidades

- Anadir restaurante
- Mostrar restaurantes
- Editar restaurante
- Eliminar restaurante
- Salir
