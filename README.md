# RipAdvisor

## Objetivos del Proyecto

Aplicacion de escritorio para gestionar un sistema de ranking de restaurantes con puntuacion de 1 a 5 estrellas.

## Requisitos del Sistema

- Java JDK 8 o superior
- Windows, Linux o macOS

## Como Ejecutar

```bash
javac -d . src/net/salesianos/*.java
java net.salesianos.App
```

## Funcionalidades

- Anadir restaurante
- Mostrar restaurantes
- Editar restaurante
- Eliminar restaurante
- Salir del programa

## Estructura de Datos

Cada restaurante tiene:
- Nombre
- Direccion
- Tipo de cocina
- Puntuacion (1-5)
- Valoracion

## Notas

- Los datos se guardan solo en memoria
- Para salir usar la opcion "Salir" del menu
