# RipAdvisor

## Descripcion

Aplicacion de escritorio para gestionar restaurantes.

## Requisitos

- Java JDK 26

## Ejecutar

javac -d . src/net/salesianos/\*.java src/App.java
java net.salesianos.App

## Opciones

- Anadir restaurante
- Mostrar restaurantes
- Editar restaurante
- Eliminar restaurante
- Salir
